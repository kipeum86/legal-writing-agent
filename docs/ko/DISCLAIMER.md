# 면책사항

## 개인 프로젝트

이 저장소는 Kipeum Lee의 **독립적인 개인 프로젝트**입니다. 어떠한 로펌, 고용주, 기관의 후원·보증·제휴와도 무관합니다. 모든 의견, 설계 결정 및 콘텐츠는 전적으로 개인의 것입니다.

## 법률 자문 아님

이 저장소의 어떠한 내용도, 그리고 에이전트가 생성한 법률문서 초안, 수정 결과물, 구조 개요, 규범 기반 서식 등 모든 산출물도 법률 자문에 해당하지 않습니다. 이러한 결과물은 **정보 제공 및 업무 보조** 목적으로만 제공됩니다. 자격을 갖춘 법률 전문가의 조언을 대체하는 것으로 신뢰하거나 의존해서는 안 됩니다.

법률문서 작성 규범은 관할권, 문서 유형, 전문 분야에 따라 상이합니다. 이 에이전트는 일반적인 서식과 구조 규범을 적용하지만, **귀하의 구체적인 관할권과 상황에 정통한 법률 전문가만이 법률문서가 사용 목적에 적합한지 보장할 수 있습니다.**

## AI의 한계 — 환각과 오류

이 프로젝트는 대규모 언어 모델(Claude)을 기반으로 합니다. LLM은 실제로 **환각, 사실 오류, 부정확한 법률 작성**을 생성할 수 있습니다. 결과물에는 다음과 같은 문제가 포함될 수 있습니다:

- 대상 관할권이나 문서 유형에 맞지 않는 법률문서 작성 규범을 적용할 수 있음
- 내장된 검증 기능에도 불구하고 일관되지 않은 용어, 번호 매기기, 상호참조를 생성할 수 있음
- 구조적으로 그럴듯하지만 내용적으로 결함이 있는 법률 문장을 생성할 수 있음
- 복잡하거나 다단계 수정 요청에서 사용자 지시를 정확히 따르지 못할 수 있음
- 사용자 의도나 적용 가능한 법적 체계를 반영하지 않는 내용을 삽입할 수 있음

이 에이전트에는 **자체 검토 파이프라인**(일관성 검사기, 규범 검증기, 용어 레지스트리)이 내장되어 있지만, 이러한 안전장치는 오류 위험을 줄일 뿐 완전히 제거하지는 못합니다. 어떠한 결과물이든 이를 근거로 행동하기 전에 **반드시 자격을 갖춘 전문가가 직접 검토하고 검증해야 합니다.**

## 인용 조작 금지 — 단, 자리 표시자는 검증 필요

이 에이전트는 **법률 인용을 절대 조작(fabricate)하지 않도록** 설계되어 있습니다. 인용이 필요하지만 사용자가 제공하지 않은 경우 `[Citation needed: 설명]` 또는 `[Authority needed: 설명]` 등의 자리 표시자를 삽입합니다. 다만:

- 사용자가 제공한 인용은 그대로 포함되며, 에이전트가 정확성을 검증하지 않습니다
- 자리 표시자는 문서 사용 전에 자격을 갖춘 전문가가 채우고 검증해야 합니다
- 에이전트의 규범 기반 인용 서식(Bluebook, OSCOLA, 한국 법률문서 규범)이 인용된 법령의 존재 여부나 현행 여부를 보장하지 않습니다

## 소스 라이브러리 및 Ingest 시스템

`/library/` 시스템과 `/ingest` 명령을 통해 외부 소스 자료(법령, 가이드라인, 판례, 학술 논문)를 에이전트의 참조 기반에 추가할 수 있습니다. 중요한 주의사항:

- **자동 분류**: Ingest 시스템은 내용 분석을 기반으로 소스를 Grade A~D로 자동 분류합니다. 이 분류는 휴리스틱(경험적 판단) 방식이며 오류가 있을 수 있습니다 — 2차 소스가 1차 소스로 잘못 분류되거나 그 반대의 경우가 발생할 수 있습니다
- **형식 변환**: 파일은 자동화된 파이프라인을 통해 Markdown으로 변환됩니다. 변환 과정에서 서식 오류가 발생하거나, 표 구조가 유실되거나, 복잡한 레이아웃이 잘못 파싱될 수 있습니다
- **독립적 검증 없음**: 에이전트는 수집된 소스를 제공된 그대로 처리합니다. 사용자가 제공한 자료의 진위, 정확성, 최신성을 검증하지 않습니다
- **Grade D 거부**: 뉴스 기사, AI 생성 요약, 위키 콘텐츠는 자동으로 거부됩니다. 다만 분류기가 간혹 저품질 소스를 수용하거나 정당한 소스를 거부할 수 있습니다

## 데이터 보안 및 프라이버시

모든 처리는 **로컬 파일시스템 내에서** 이루어지며, 외부 서버나 데이터베이스를 사용하지 않습니다. 다만 에이전트는 LLM API를 호출하므로 **문서 텍스트와 사용자 지시사항이 모델 제공자**(Anthropic)**에게 전송됩니다**.

기밀 문서를 포함한 실제 법률 업무에 이 도구를 사용하려는 경우:

- **최소한**, 무료 티어나 소비자용 플랜이 아닌 **상용 API 키**를 사용하여 표준 상용 데이터 처리 약관이 적용되도록 하세요.
- **이상적으로는**, **Zero Data Retention (ZDR) 계약이 포함된 Enterprise 플랜**을 사용하세요. 단, ZDR이 **완전한 보호를 보장하지는 않습니다**. 데이터는 여전히 제공자 인프라를 경유합니다. 조직 요건에 부합하는지 판단하려면 Anthropic의 [ZDR 및 데이터 처리 문서](https://code.claude.com/docs/en/zero-data-retention)를 직접 확인하세요.
- **반드시**, 기밀정보·특권정보·개인정보를 어떠한 AI 도구로든 처리하기 전에 조직의 **보안·컴플라이언스·IT 부서**와 사전에 협의하세요.

## 책임 범위

작성자는 다음 사항에 대해 책임을 지지 않습니다:

- 에이전트 산출물을 근거로 이루어진 모든 의사결정
- 생성된 콘텐츠에 의존하여 발생한 손실, 법적 결과 및 손해
- 산출물의 정확성, 완전성 및 특정 목적 적합성
- API 키, 네트워크 설정 또는 사용자 과실로 인한 데이터 노출
- 이 에이전트가 생성한 문서의 법적 유효성 또는 집행 가능성
- 소스 수집, 형식 변환 또는 등급 분류 과정에서 발생한 오류
