# Formatting Modes Reference

Mode A-D guidance is optional reference material. Load this file only when the user requests one of these output modes or the selected template explicitly depends on it.

## Mode A — Executive Brief

Use for short decision-maker outputs. Compress analysis, foreground key conclusions, keep citations sparse and anchored, and target 2-4 pages. Do not use Mode A as the default for ordinary corporate or advisory drafts.

## Mode B — Comparative Matrix

Use for cross-jurisdiction comparisons. The matrix is the main analytical surface; every populated cell needs a source. Do not load for single-jurisdiction drafting.

## Mode C — Enforcement and Case Law

Use for enforcement timelines, case-law development, and decision summaries. Citation density is high and chronology matters. Do not load unless the user asks for enforcement/case-law treatment or provides a case packet.

## Mode D — Black-letter and Commentary

Use for article-by-article statute commentary. It is the heaviest mode and should be selected only for legislation-focused analysis with an authority packet.

## Drafting-Scope Reminder

Mode selection changes format and density only. It does not authorize independent legal conclusions, risk assessments, recommendations, certainty levels, or legal research. Apply `docs/policies/drafting-scope.md` first.
