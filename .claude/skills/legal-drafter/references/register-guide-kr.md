# Korean Legal Register Guide (한국 법률 문체 종합 가이드)

이 문서는 한국어 법률 문서 작성 시 준수해야 할 문체, 표현, 형식 규칙을 종합적으로 정리한 참조 가이드이다.

---

## 1. Document-Type Register Matrix (문서 유형별 문체 기준)

| 문서 유형 | 격식 수준 | 종결어미 | 비고 |
|---|---|---|---|
| 법령/조례 | 최고 격식 (문어체) | ~한다, ~로 한다, ~하여야 한다 | 가장 엄격한 문어체. 예외 없음. |
| 정관/사규 | 최고 격식 (문어체) | ~한다, ~로 한다, ~하여야 한다 | 법령과 동일한 수준 적용. |
| 이사회 결의서 | 고격식 | ~한다, ~하기로 하다, ~을(를) 승인하다 | 결의 사항은 "~하기로 하다" 형식 사용. |
| 주주총회 의사록 | 고격식 | ~하였다, ~라고 보고하다, ~하기로 의결하다 | 과거 시제 중심. 보고·의결 구분 명확히. |
| 소장/답변서 | 격식 (경어체) | ~하였습니다, ~입니다 | 법원 제출 문서. "~습니다" 체 사용. |
| 준비서면 | 격식 (경어체 또는 문어체) | ~하였습니다, ~입니다 또는 ~한다 | 법원별·재판부별 관행 차이 존재. |
| 법률의견서 | 격식 (경어체) | ~합니다, ~사료됩니다, ~판단됩니다, ~것입니다 | 의뢰인 대상. 전문성과 예우를 겸비. |
| 내부 메모 | 준격식 | ~합니다 체 | 상대적으로 덜 격식적이나 구어체 금지. |
| 정책문서/가이드라인 | 격식 (혼용 가능) | ~한다 체 또는 ~합니다 체 | 대상 독자에 따라 선택. 일관성 유지 필수. |

### 문서 유형별 적용 원칙

- 하나의 문서 내에서 종결어미 체계를 혼용하지 않는다.
- 법원 제출 문서는 해당 재판부의 관행을 우선적으로 확인한다.
- 의뢰인 대상 문서는 "~합니다" 체를 기본으로 하되, 인용 조문 설명 시 "~한다"를 병기할 수 있다.

## 2. Sentence Endings (종결어미)

### 2.1 법령·규정·결의서용 (Mandatory 문어체)

| 패턴 | 용례 |
|---|---|
| ~한다 | 이사회는 다음 사항을 결의한다. |
| ~하여야 한다 | 감사는 이사의 직무집행을 감사하여야 한다. |
| ~할 수 있다 | 주주는 서면으로 의결권을 행사할 수 있다. |
| ~로 한다 | 본 회사의 상호는 주식회사 OO로 한다. |
| ~에 해당한다 | 이는 상법 제382조에 해당한다. |
| ~하지 못한다 | 이사는 이사회의 승인 없이 거래하지 못한다. |
| ~에 의한다 | 이사의 선임은 주주총회 결의에 의한다. |
| ~하지 아니한다 | 본 조항은 소급하여 적용하지 아니한다. |
| ~을(를) 요한다 | 정관 변경은 특별결의를 요한다. |

### 2.2 의견서·보고서·법원 제출 문서용 (경어체)

| 패턴 | 용례 |
|---|---|
| ~합니다 | 검토 결과를 아래와 같이 보고합니다. |
| ~됩니다 | 본 건은 상법 제382조가 적용됩니다. |
| ~것입니다 | 위 해석이 타당할 것입니다. |
| ~사료됩니다 | 원고의 청구가 인용될 가능성이 높은 것으로 사료됩니다. |
| ~판단됩니다 | 본 거래는 이해충돌에 해당하는 것으로 판단됩니다. |
| ~하였습니다 | 피고는 2025. 3. 15. 계약을 해제하였습니다. |
| ~바 있습니다 | 대법원은 이와 유사한 사안에서 판시한 바 있습니다. |

### 2.3 의사록 전용 (보고·의결 구조)

| 패턴 | 용례 |
|---|---|
| ~라고 보고하다 | 대표이사는 영업 실적을 아래와 같이 보고하였다. |
| ~하기로 의결하다 | 제1호 안건을 원안대로 승인하기로 의결하였다. |
| ~하기로 하다 | 배당금을 1주당 금 500원으로 하기로 하였다. |

## 3. Formal Connectors 완전판 (접속사/접속부사)

### 3.1 인과 (Causal)

| 접속어 | 용법 | 용례 |
|---|---|---|
| 따라서 | 결론 도출 시 | 따라서 피고의 주장은 이유 없다. |
| 그러므로 | 보다 격식적인 결론 | 그러므로 원고의 청구는 기각되어야 한다. |
| 이에 | 앞 내용에 근거하여 | 이에 본 건 결의를 상정한다. |
| 그 결과 | 사실관계의 결과 | 그 결과 손해액은 금 1억 원에 달하였다. |
| 이로 인하여 | 원인-결과 연결 | 이로 인하여 원고는 상당한 손해를 입었다. |
| ~에 기인하여 | 원인 특정 | 피고의 과실에 기인하여 사고가 발생하였다. |

### 3.2 역접 (Adversative)

| 접속어 | 용법 | 용례 |
|---|---|---|
| 다만 | 제한·예외 조건 | 다만, 선의의 제3자에게는 대항하지 못한다. |
| 그러나 | 일반적 역접 | 그러나 이 해석에는 의문이 있다. |
| 한편 | 대비·전환 | 한편, 피고 측은 이와 다른 입장이다. |
| 그럼에도 불구하고 | 강한 역접 | 그럼에도 불구하고 피고는 이행을 거부하였다. |
| 이에 반하여 | 대립 의견·사실 | 이에 반하여 소수설은 무효라고 본다. |
| 반면에 | 대조 | 반면에 을 회사의 매출은 감소하였다. |

### 3.3 추가 (Additive)

| 접속어 | 용법 | 용례 |
|---|---|---|
| 또한 | 일반적 추가 | 또한 상법 제393조도 검토할 필요가 있다. |
| 아울러 | 병행 검토 | 아울러 세법상 쟁점도 살펴본다. |
| 나아가 | 논의 확장 | 나아가 헌법적 쟁점까지 검토할 필요가 있다. |
| 더불어 | 병행 사항 | 더불어 내부통제 절차의 정비도 권고한다. |
| 이에 더하여 | 추가 근거 | 이에 더하여 다음 사정도 고려하여야 한다. |
| 뿐만 아니라 | 추가 강조 | 뿐만 아니라 형사책임의 문제도 발생할 수 있다. |

### 3.4 조건 (Conditional)

| 접속어 | 용법 | 용례 |
|---|---|---|
| ~하는 경우 | 일반 조건 | 주주가 반대하는 경우 매수청구권을 행사할 수 있다. |
| ~한 때에는 | 법령 조건문 | 이사가 법령을 위반한 때에는 손해를 배상하여야 한다. |
| ~하는 한 | 한정 조건 | 선관주의의무를 다하는 한 책임을 지지 아니한다. |
| ~을(를) 전제로 | 전제 조건 | 이사회 승인을 전제로 계약을 체결할 수 있다. |

### 3.5 예시 (Exemplification)

| 접속어 | 용법 | 용례 |
|---|---|---|
| 예컨대 | 구체적 예시 | 예컨대, 자기거래가 이에 해당한다. |
| 이를테면 | 예시 (약간 덜 격식) | 이를테면, 주주간 계약이 그러하다. |
| 가령 | 가정적 예시 | 가령 대표이사가 사임하는 경우를 상정할 수 있다. |
| 구체적으로 | 구체화 | 구체적으로 살펴보면 다음과 같다. |

### 3.6 정리·결론 (Summary)

| 접속어 | 용법 | 용례 |
|---|---|---|
| 요컨대 | 핵심 요약 | 요컨대, 본 거래는 적법하다. |
| 결국 | 최종 결론 | 결국, 원고의 청구는 인용되어야 한다. |
| 종합하면 | 종합 정리 | 종합하면, 본 안건은 승인이 타당하다. |
| 이상을 종합하건대 | 장문 결론 (소장·준비서면) | 이상을 종합하건대, 피고의 항변은 이유 없다. |

### 3.7 전환 (Transition)

| 접속어 | 용법 | 용례 |
|---|---|---|
| 한편 | 화제 전환 | 한편, 세법상 쟁점도 별도로 존재한다. |
| 다른 한편으로 | 대안적 관점 | 다른 한편으로, 실무적 리스크도 고려하여야 한다. |
| 이와 별도로 | 별개 사항 | 이와 별도로, 공정거래법상 문제도 검토한다. |
| 이와 관련하여 | 관련 사항 연결 | 이와 관련하여, 대법원 판례를 살펴본다. |

### 3.8 강조 (Emphasis)

| 접속어 | 용법 | 용례 |
|---|---|---|
| 특히 | 특별 강조 | 특히, 이해충돌 방지 의무에 주의하여야 한다. |
| 무엇보다 | 최우선 강조 | 무엇보다 주주의 이익 보호가 중요하다. |
| 주목할 점은 | 독자 주의 환기 | 주목할 점은, 판례가 변경되었다는 것이다. |
| 중요한 것은 | 핵심 사항 강조 | 중요한 것은 계약서의 문언이다. |

## 4. Prohibited Patterns (금지 표현 확장)

| 금지 표현 | 교정 | 비고 |
|---|---|---|
| ~거든요 | ~이므로 / ~이기 때문에 | 구어체 이유 제시 |
| ~잖아요 | ~인바 / ~인 점에 비추어 | 구어체 확인 요구 |
| ~인데요 | ~인바 / ~인데 | 구어체 상황 설명 |
| ~같아요 | ~로 판단된다 / ~로 사료된다 | 구어체 추측 |
| ~할 거예요 | ~할 것이다 / ~할 예정이다 | 구어체 미래 표현 |

| 약간 | 다소 / 일부 | 정도 표현 |
| 엄청 | 극히 / 현저히 | 정도 강조 |
| 완전 | 전적으로 / 완전히 | 완전성 표현 |

### 4.4 번역투 (외국어 직역체)

| 금지 표현 | 교정 | 비고 |
|---|---|---|
| ~에 의해 ~되어지다 | ~에 의하여 ~되다 / ~로 ~하다 | 영어 수동태 직역 |

| ~를 가지다 | ~이(가) 있다 | "have" 직역 |
| ~에 대한 ~을 하다 | ~을(를) ~하다 | 불필요한 "에 대한" 삽입 |
| ~하는 것으로 되어 있다 | ~하도록 규정하고 있다 | 불명확한 피동 표현 |

## 6. Legal Honorifics and Formality Levels (법률 경칭 및 호칭)

### 6.1 법원에 대한 경칭

| 표현 | 사용 맥락 |
|---|---|
| 귀원 | 소장·준비서면에서 법원 지칭 |
| 귀 법원 | 소장·준비서면에서 법원 지칭 (대안) |
| 귀하 | 상대방에 대한 일반 경칭 (통지서 등) |

### 6.2 당사자 지칭

| 표현 | 사용 맥락 |
|---|---|
| 원고 / 피고 | 법원 제출 문서에서 소송 당사자 |
| 채권자 / 채무자 | 집행·도산 절차 문서 |
| 신청인 / 피신청인 | 가처분·중재 신청 문서 |
| 귀하 | 의견서·통지서에서 상대방 |
| 귀사 | 기업 대상 의견서·통지서 |
| 당사 | 자기 회사 지칭 (의견서·계약서) |
| 본인 | 자연인 자기 지칭 |

### 6.3 법관 호칭

| 표현 | 사용 맥락 |
|---|---|
| 재판장 | 재판장 판사 호칭 |
| 판사 | 일반 판사 호칭 |
| 재판부 | 합의부 전체 지칭 |

### 6.4 주의사항

- 법원 제출 문서에서 "저" 또는 "나"를 사용하지 않는다. "원고" 또는 "피고"로 지칭한다.
- 의뢰인 대상 의견서에서 의뢰인을 "귀하" 또는 "귀사"로 지칭한다.
- 상대방 대리인을 지칭할 때는 "상대방 대리인" 또는 "피고 소송대리인"을 사용한다.

## 7. Number and Date Conventions (숫자·날짜·금액 표기)

### 7.1 금액 표기

| 형식 | 용례 | 비고 |
|---|---|---|
| 금 OOO원 | 금 123,000,000원 | 기본 형식 |
| 한글 병기 | 금 일억이천삼백만원(₩123,000,000) | 계약서·판결문 등 정확성 요구 시 |
| 외화 표기 | 미합중국 통화 10,000달러(USD 10,000) | 외화 금액 |

### 7.2 비율 표기

| 올바른 표기 | 금지 표기 |
|---|---|
| 연 5% | 5프로 |
| 월 0.5% | 0.5프로 |
| 연 12%의 비율에 의한 금원 | 12프로 |

### 7.3 날짜 표기

| 형식 | 용례 | 사용 맥락 |
|---|---|---|
| YYYY. MM. DD. | 2025. 3. 15. | 법원 제출 문서 (마침표 포함) |
| YYYY년 MM월 DD일 | 2025년 3월 15일 | 계약서·정관 등 |
| YYYY/MM/DD | (사용 금지) | 법률 문서에서 부적절 |
| YYYY.MM.DD | (사용 금지) | 법률 문서에서 부적절 |

### 7.4 기간 표기

| 올바른 표기 | 금지 표기 |
|---|---|
| N일 | N일간 (문맥에 따라 허용) |
| N개월 | N달 |
| N년 | N년간 (문맥에 따라 허용) |

### 7.5 조문 번호 표기

| 올바른 표기 | 금지 표기 |
|---|---|
| 제1조 | 1조 |
| 제2항 | 2항 (단, 약칭 관행에 따라 허용되는 경우 있음) |
| 제3호 | 3호 |
| 제1조 제2항 제3호 | 1조 2항 3호 |
| 제1조의2 | 제1조 의 2 |

## 8. Punctuation in Korean Legal Documents (법률 문서 문장부호)

### 8.1 기본 문장부호

| 부호 | 용법 | 용례 |
|---|---|---|
| . (마침표) | 문장 끝에 반드시 사용 | 이사회는 이를 승인한다. |
| , (쉼표) | 병렬 구조, 조건절 뒤, 부사절 뒤 | 다만, 선의의 제3자에게는 대항하지 못한다. |

### 8.2 법률 전용 문장부호

| 부호 | 용법 | 용례 |
|---|---|---|
| 「」 (낫표) | 법률명 인용 | 「상법」 제382조 |
| 『』 (겹낫표) | 하위 법령·규정 인용 | 『상법 시행령』 제12조 |
| " " (큰따옴표) | 정의 용어 표시 | "이사"라 함은 이사회의 구성원을 말한다. |
| ( ) (소괄호) | 부연 설명, 영문 병기, 약칭 정의 | 주식회사 OO(이하 "회사"라 한다) |
| · (가운뎃점) | 병렬 나열 | 갑·을·병 |

### 8.3 사용 주의사항

- 법률명 인용 시 「」를 일관되게 사용한다. 《》는 사용하지 않는다.
- 쉼표는 접속부사(다만, 한편, 또한 등) 뒤에 사용한다.
- 마침표 없이 문장을 끝내지 않는다.
- 조문 나열 시: 제1조, 제2조 및 제3조 (마지막 항목 전에 "및" 사용).
- 목록 나열 시: 1. 2. 3. 또는 가. 나. 다. 형식을 사용한다.

## 9. Spacing Rules (띄어쓰기 주의사항)

### 9.1 자주 틀리는 띄어쓰기

| 올바름 (O) | 오류 (X) | 규칙 |
|---|---|---|
| 수밖에 | 수 밖에 | "밖에"는 조사이므로 붙여 쓴다 |
| 할 수 있다 | 할수있다 | "수"는 의존명사이므로 띄어 쓴다 |
| 뿐만 아니라 | 뿐 만 아니라 | "뿐만"은 붙여 쓴다 |
| 및 | ~및~ (앞뒤 무띄어쓰기) | "및" 앞뒤로 띄어쓰기 |
| 등 | ~등 (앞 무띄어쓰기) | "등" 앞에 띄어쓰기 |
| ~에 따라 | ~에따라 | 조사+동사 사이 띄어쓰기 |
| ~할 뿐이다 | ~할뿐이다 | "뿐"은 의존명사이므로 띄어 쓴다 |
| ~한 바 있다 | ~한바있다 | "바"는 의존명사이므로 띄어 쓴다 |
| ~할 것이다 | ~할것이다 | "것"은 의존명사이므로 띄어 쓴다 |
| ~하는 데 | ~하는데 (구별 필요) | 의존명사 "데"는 띄어 쓰나, 어미 "-는데"는 붙여 쓴다 |
| ~함에 있어 | ~함에있어 | 조사+명사 사이 띄어쓰기 |

### 9.2 법률 용어 띄어쓰기

| 올바름 (O) | 오류 (X) |
|---|---|
| 주식회사 | 주식 회사 |
| 이사회 | 이사 회 |
| 주주총회 | 주주 총회 |
| 대표이사 | 대표 이사 |
| 감사위원회 | 감사 위원회 |
| 손해배상 | 손해 배상 (문맥에 따라 띄어쓰기 가능) |
| 채무불이행 | 채무 불이행 (문맥에 따라 띄어쓰기 가능) |

## 10. Quick Reference Checklist (작성 전 점검표)

문서 작성 완료 후 다음 사항을 점검한다:

1. [ ] 종결어미가 문서 유형에 맞는 체계로 일관되게 사용되었는가?
2. [ ] 구어체 표현(종결어미, 부사, 접속사)이 포함되어 있지 않은가?
3. [ ] 번역투·이중 피동 표현이 사용되지 않았는가?
4. [ ] 법률명 인용 시 「」를 사용하였는가?
5. [ ] 금액·날짜·조문 번호 표기가 올바른가?
6. [ ] 띄어쓰기 규칙을 준수하였는가?
7. [ ] 경칭·호칭이 문서 유형과 대상에 적합한가?
8. [ ] 문장부호(마침표, 쉼표)가 올바르게 사용되었는가?
9. [ ] 하나의 문서 내에서 종결어미 체계가 혼용되지 않았는가?
10. [ ] 불필요하게 장황한 표현이 없는가?
