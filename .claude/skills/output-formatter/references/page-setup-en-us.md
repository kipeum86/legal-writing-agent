# US English Document Page Setup

## Scope
Use this page setup for US English outputs unless a court rule, agency rule, local rule, filing template, or house style provides stricter requirements.

## US Letter Standard
| Parameter | Value |
|---|---|
| Page size | US Letter (8.5″ × 11″ / 215.9mm × 279.4mm) |
| Top margin | 1″ (25.4mm) |
| Bottom margin | 1″ |
| Left margin | 1″ |
| Right margin | 1″ |
| Header distance | 0.5″ |
| Footer distance | 0.5″ |

## Typography
| Element | Font | Size | Weight |
|---|---|---|---|
| Body text | Times New Roman | 12pt | Regular |
| Heading 1 | Times New Roman or Arial | 14pt | Bold |
| Heading 2 | Times New Roman or Arial | 13pt | Bold |
| Heading 3 | Times New Roman or Arial | 12pt | Bold |
| Caption | Times New Roman | 10pt | Regular |
| Footnote | Times New Roman | 10pt | Regular |

## Line Spacing
| Document type | Spacing |
|---|---|
| Court briefs | Double-spaced unless a rule specifies otherwise |
| Legal memoranda | 1.5 |
| Corporate documents | 1.15–1.5 |
| Footnotes | Single |
| Block quotations | Single or 1.15 |

## Paragraph Settings
- First-line indent: 0.5″ for briefs and memoranda unless the template uses block paragraphs
- Paragraph spacing after: 12pt for business-style prose, or 0pt when using full double spacing
- Alignment: left-aligned for memoranda; justified only if the template or house style requires it
- Use hanging indentation for numbered lists and nested provisions

## Headings and Numbering
- Main titles may be centred; analytical headings are usually left-aligned
- Court captions should follow the relevant court's rule set, not generic memo formatting
- Decimal numbering and `(a)`, `(i)` nesting should use consistent indentation across the document
- Avoid mixing `Article` and `Section` unless the template requires both

## Footnotes, Quotations, and Tables
- Footnotes should remain 10pt, single-spaced
- Block quotations should be indented and visually separated from body text
- Table text may be reduced to 10pt or 11pt where necessary for fit, but body prose should remain 12pt
- Keep borders and shading conservative

## Headers, Footers, and Pagination
- Use page numbers consistently
- Running headers should be matter-appropriate and restrained
- Do not add decorative elements unless required by house style
- Filing documents should reserve space for court stamps or docketing marks if the forum requires it

## Signature Blocks
- Leave 18pt to 24pt above signature lines
- Include signatory name, title, and date as required by the document type
- For filings, include bar number and service/contact details where required
