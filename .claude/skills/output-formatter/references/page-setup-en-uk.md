# UK/International English Document Page Setup

## A4 Standard
| Parameter | Value |
|---|---|
| Page size | A4 (210mm × 297mm) |
| Top margin | 2.54cm (1″) |
| Bottom margin | 2.54cm |
| Left margin | 2.54cm |
| Right margin | 2.54cm |

## Typography
| Element | Font | Size | Weight |
|---|---|---|---|
| Body text | Times New Roman | 12pt | Regular |
| Heading 1 | Arial or Times New Roman | 14pt | Bold |
| Heading 2 | Arial or Times New Roman | 13pt | Bold |
| Heading 3 | Arial or Times New Roman | 12pt | Bold |

## Line Spacing
- Default: 1.5 spacing
- Court submissions: per court rules

## Paragraph
- First line indent: none (UK style) or 1cm
- Paragraph spacing: 6–12pt after
- Alignment: Justified
