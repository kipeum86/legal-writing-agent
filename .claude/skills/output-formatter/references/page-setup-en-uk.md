# UK/International English Document Page Setup

## Scope
Use this page setup for UK English and neutral international English outputs unless the user, tribunal, court, or house style supplies stricter formatting instructions.

## A4 Standard
| Parameter | Value |
|---|---|
| Page size | A4 (210mm × 297mm) |
| Top margin | 2.54cm (1″) |
| Bottom margin | 2.54cm |
| Left margin | 2.54cm |
| Right margin | 2.54cm |
| Header distance | 1.25cm |
| Footer distance | 1.25cm |

## Typography
| Element | Font | Size | Weight |
|---|---|---|---|
| Body text | Times New Roman | 12pt | Regular |
| Heading 1 | Arial or Times New Roman | 14pt | Bold |
| Heading 2 | Arial or Times New Roman | 13pt | Bold |
| Heading 3 | Arial or Times New Roman | 12pt | Bold |
| Footnotes | Times New Roman | 10pt | Regular |
| Captions / labels | Times New Roman | 10pt | Regular |

## Line Spacing
| Document type | Spacing |
|---|---|
| Advice memoranda | 1.5 |
| Corporate / policy documents | 1.15–1.5 |
| Court or tribunal submissions | Follow forum rules first; if unspecified, use 1.5 |
| Footnotes | Single |
| Block quotations | Single or 1.15 |

## Paragraph Settings
- First-line indent: none for most UK and international business/legal prose
- Paragraph spacing after: 6pt to 12pt
- Alignment: justified by default; left-aligned if house style or tribunal rules require it
- Keep one numbering style per document; do not hand-mix hanging and non-hanging numbering layouts

## Headings and Numbering
- Heading 1 normally aligns left unless the document template requires centred titles
- Decimal numbering should use consistent hanging indentation
- Clause-based documents should align wrapped text under the clause text, not under the number alone
- Lettered subparagraphs `(a)`, `(b)`, `(c)` should use hanging indentation for readability

## Footnotes and Citations
- Use footnotes for OSCOLA-driven authority references
- Footnote text should be 10pt, single-spaced
- Separate footnotes with 0pt or 3pt after-spacing, not full paragraph spacing
- Do not insert large blank lines before the footnote separator

## Tables, Quotations, and Appendices
- Table body text: 10pt to 11pt
- Keep table borders light and conservative
- Block quotations: indent left and right, reduce spacing, and omit quotation marks for longer extracts where house style permits
- Appendices / schedules should start on a new page when they are substantial

## Headers, Footers, and Pagination
- Use page numbers in the footer or header per house style
- Running headers should be short and descriptive
- Avoid decorative elements unless the house style expressly requires them
- International documents should avoid jurisdiction-specific court caption styling unless required by the forum

## Signature Blocks
- Leave at least 18pt to 24pt above signature lines
- Use title, organisation, and date lines where appropriate
- For letters and submissions, align signature blocks consistently with the document template
