# UK English Legal Writing Style Guide (style-guide-en-uk)

## Key Differences from US

| Feature | UK Convention |
|---|---|
| Terminology | "Clause" not "Section"; "Schedule" not "Exhibit" |
| Page size | A4 |
| Citation | OSCOLA (Oxford Standard for Citation of Legal Authorities) |
| Date | DD Month YYYY (e.g., 10 March 2026) |
| Spelling | UK English (honour, defence, licence/license distinction) |
| Numbering | Clause 1, 1.1, 1.1.1 or (a), (b), (c) |
| Margins | 2.54cm (1″) all sides |
| Font | Times New Roman, 12pt |

## OSCOLA Citation

### Cases
```
Party v Party [Year] Court Reference
```
Example: R v Smith [2023] UKSC 45

### Statutes
```
Act Name Year, s Section
```
Example: Companies Act 2006, s 171

### Secondary Sources
Footnote-based system. No in-text citations.

## Register
- Formal but less archaic than traditional drafting
- "Must" preferred over "shall" (following modern drafting guidance)
- Plain English encouraged per Clarity movement
