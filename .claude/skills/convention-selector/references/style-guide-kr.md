# Korean Legal Writing Style Guide (style-guide-kr)

---

## 1. Register (문체)

### Formal Written Register (문어체)
- Default for all legal documents
- Sentence endings: ~한다, ~하여야 한다, ~할 수 있다, ~로 한다, ~에 해당한다
- Advisory/memo exception: ~합니다 체 permitted when appropriate

### Document-Type Specific Sentence Endings

| 문서 유형 | 주요 어미 | 예시 |
|---|---|---|
| 법령·정관·규정 | ~한다, ~로 한다, ~아니한다 | "이사의 임기는 3년으로 한다." |
| 소장·준비서면 | ~하였습니다, ~입니다, ~바입니다 | "원고는 피고에게 금 1억 원의 지급을 구합니다." |
| 의견서·법률 메모 | ~합니다, ~사료됩니다, ~판단됩니다 | "본건 계약은 유효한 것으로 판단됩니다." |
| 결의서 | ~한다, ~하기로 하다, ~의결하다 | "다음 사항을 의결한다." |
| 계약서 | ~한다, ~로 한다, ~기로 한다 | "갑은 을에게 대금을 지급한다." |
| 통지서·내용증명 | ~합니다, ~하시기 바랍니다 | "기한 내에 이행하시기 바랍니다." |

### Common Register Mistakes and Corrections

| 오류 | 수정 | 설명 |
|---|---|---|
| 정관에서 "~합니다" 사용 | "~한다"로 수정 | 정관은 규범적 문체 사용 |
| 소장에서 "~한다" 사용 | "~합니다"로 수정 | 법원 제출 서면은 경어체 사용 |
| 같은 문서에서 "~한다"와 "~합니다" 혼용 | 하나로 통일 | 문체 일관성 유지 |
| 의견서에서 "~인 것 같습니다" | "~로 판단됩니다" | 불확실한 표현 지양 |

### Prohibited
- Colloquial expressions (구어체)
- Contractions or abbreviations (unless established legal abbreviations)
- Informal sentence endings (~거든요, ~잖아요, ~인데요)
- Emotional or subjective expressions (~참으로 안타까운, ~도저히 납득할 수 없는)

## 2. Numbering System (조항호목)

```
제1조 (제목)
  ① 제1항 내용
    1. 제1호 내용
      가. 목 내용
      나. 목 내용
    2. 제2호 내용
  ② 제2항 내용

제2조 (제목)
  ...
```

### Rules
- 조(條): Top-level divisions. 제1조, 제2조, 제3조...
- 항(項): Sub-divisions of 조. ①, ②, ③... (circled numbers)
- 호(號): Enumerated items within 항. 1., 2., 3...
- 목(目): Sub-items within 호. 가., 나., 다., 라...

### For Non-Statutory Documents
Advisory, corporate, and general documents may use:
```
I. 대제목
  1. 중제목
    가. 소제목
      (1) 세부사항
        (가) 세부세부사항
```

## 3. Citation Format

### Statutes (법령 인용)
```
「법률명」 제N조 제N항 제N호
```
Examples:
- 「민법」 제750조
- 「상법」 제382조 제2항
- 「자본시장과 금융투자업에 관한 법률」(이하 "자본시장법") 제9조 제1항

### Court Decisions (판례 인용)
```
대법원 YYYY. MM. DD. 선고 YYYY다NNNNN 판결
```
Examples:
- 대법원 2023. 5. 18. 선고 2022다12345 판결
- 헌법재판소 2022. 12. 22. 선고 2020헌바123 결정

### First Reference Rule
- Full citation on first reference
- Abbreviated form thereafter: "위 판결", "같은 법 제N조"

## 8. Defined Terms
- First use: full name with defined abbreviation
- Pattern: "주식회사 OO(이하 "갑"이라 한다)"
- Subsequent uses: defined abbreviation only ("갑")

## 10. Common Korean Legal Writing Errors (자주 발생하는 오류)

### 10.1 주어-서술어 호응 오류

| 오류 (비문) | 수정 | 설명 |
|---|---|---|
| "원고의 주장은 타당한 것으로 사료됩니다." → 주어와 서술어 불일치 시 | "원고의 주장은 타당합니다." 또는 "원고의 주장이 타당한 것으로 사료됩니다." | 주어에 맞는 서술어를 사용 |
| "본 계약의 목적은 갑이 을에게 용역을 제공하여 그 대가를 지급한다." | "본 계약의 목적은 갑이 을에게 용역을 제공하고 그 대가를 지급받는 것이다." | "목적은"에는 "~것이다"가 호응 |
| "위 사실에 비추어 피고가 원고에게 손해배상 의무를 부담한다." | "위 사실에 비추어 보면, 피고는 원고에게 손해배상 의무를 부담한다." | "비추어"에는 "보면"이 호응 |

### 10.2 조사 오류

| 오류 | 수정 | 규칙 |
|---|---|---|
| "회사에 통지하다" (사람 대상) | "회사에게 통지하다" | 사람·법인 = 에게, 장소·시간 = 에 |
| "법률에 의거하여" (받침 없는 명사) | "법률에 의거하여" (맞음) / "규정에 의거하여" | "에 의거하여"는 올바른 표현 |
| "제도와 정책" (받침 없는 명사 뒤) | "제도와 정책" (맞음) | 받침 없음 = 와, 받침 있음 = 과 |
| "현금으로 지급" (받침 있는 명사 뒤) | "현금으로 지급" (맞음) | ㄹ 받침 또는 받침 없음 = 로, 그 외 받침 = 으로 |

### 10.3 이중 피동

| 오류 | 수정 | 설명 |
|---|---|---|
| 되어지다 | 되다 | "계약이 체결되어진다" → "계약이 체결된다" |
| 보여지다 | 보이다 | "타당하게 보여진다" → "타당하게 보인다" |
| 써여지다 | 쓰이다 | "문서가 작성되어져야 한다" → "문서가 작성되어야 한다" |
| 만들어지다 | 만들다/이루어지다 | "합의가 만들어지다" → "합의가 이루어지다" |
| 읽혀지다 | 읽히다 | 이중 피동 제거 |

### 10.4 이중 부정 혼란

| 표현 | 의미 | 더 명확한 대체 |
|---|---|---|
| "않을 수 없다" | 긍정 (해야 한다) | "~하여야 한다" 또는 "~할 수밖에 없다" |
| "없지 않다" | 긍정 (있다) | "~가 있다" |
| "아니라고 할 수 없다" | 긍정 | 직접 긍정 표현 사용 |
| "하지 않을 수 없었다" | 과거 긍정 | "~하였다" 또는 "~할 수밖에 없었다" |

> 법률 문서에서는 이중 부정을 가급적 피하고 직접적인 표현을 사용한다. 다만, 수사적으로 강조가 필요한 경우에 한하여 제한적으로 사용한다.

### 10.5 불필요한 한자어 남용

| 불필요한 한자어 | 쉬운 대체 | 비고 |
|---|---|---|
| 차(此)에 | 이에 | 공문서 순화 |
| 당해(當該) | 해당, 그 | "당해 계약" → "해당 계약" |
| 상기(上記) | 위, 앞서 말한 | "상기 내용" → "위 내용" |
| 하기(下記) | 아래, 다음 | "하기 사항" → "아래 사항" |
| 기(旣) 체결된 | 이미 체결된 | 일본식 한자어 |
| 즉(卽) | 곧, 다시 말해 | 문맥에 따라 선택 |

### 10.6 번역투 표현 (영어 직역체)

| 번역투 | 자연스러운 표현 | 원인 |
|---|---|---|
| "~하는 것이 가능하다" | "~할 수 있다" | "it is possible to" 직역 |
| "~에 의해 수행되다" | "~이(가) 수행하다" | 수동태 직역 |
| "~의 경우에 있어서" | "~하는 경우(에)" | "in the case of" 직역 |
| "~와 관련하여" (남용) | 구체적 조사 사용 | "in relation to" 직역 |
| "~라는 사실에도 불구하고" | "~하더라도" 또는 "~하지만" | "despite the fact that" 직역 |
| "~하기 위한 목적으로" | "~하기 위하여" 또는 "~하려고" | "for the purpose of" 직역 |
| "그것은 ~이다" | 주어를 구체적으로 표시 | "it is" 직역 |

### 10.7 중첩 소유격 ("~의 ~의 ~의")

| 오류 | 수정 | 방법 |
|---|---|---|
| "갑의 본건 계약의 위반의 효과" | "갑이 본건 계약을 위반한 경우의 효과" | 조사 변환, 절로 풀기 |
| "회사의 이사회의 결의의 내용" | "회사 이사회가 결의한 내용" 또는 "회사 이사회 결의 내용" | "의" 생략 또는 절로 변환 |
| "원고의 재산의 처분의 제한" | "원고 재산에 대한 처분 제한" | "의" 생략 및 적절한 조사 사용 |

> "의"가 연속 3회 이상 나오면 반드시 문장을 다듬는다.

### 10.8 비문 (주어 없는 문장, 서술어 누락)

| 유형 | 오류 예시 | 수정 |
|---|---|---|
| 주어 누락 | "상기 규정에 의하면 허용되지 아니한다." | "상기 규정에 의하면, 이러한 행위는 허용되지 아니한다." |
| 서술어 누락 | "을이 갑에게 지급하여야 할 대금 및 그 이자." | "을이 갑에게 대금 및 그 이자를 지급하여야 한다." |
| 문장 중도 종결 | "원고가 주장하는 바와 같이, 피고의 의무 위반으로 인하여." | "원고가 주장하는 바와 같이, 피고는 의무를 위반하였다." |

### 10.9 기타 자주 혼동하는 표현

| 오류 또는 혼동 | 올바른 용법 | 설명 |
|---|---|---|
| "~에 대하여" 남용 | "~을(를)", "~에 관하여" 등으로 다양화 | "에 대하여"가 만능 조사가 아님. 문맥에 맞는 조사를 사용 |
| "~하는 바" 오용 | "~하는 바, ~" (뒤에 결과·이유가 이어져야 함) | 독립적으로 사용 불가 |
| "~에 있어서" 남용 | "~에서", "~의 경우" 등으로 대체 | 일본어 투 (において) |
| "~로서" vs "~로써" | 로서 = 자격·지위, 로써 = 수단·도구 | "대표이사로서 서명하다" (자격), "서면으로써 통지하다" (수단) |
| "~므로" vs "~(으)므로" | 받침 없는 동사 뒤 = ~므로, 받침 있는 동사 뒤 = ~으므로 | "하므로" (O), "있으므로" (O) |
| "~던" vs "~든" | ~던 = 과거 회상, ~든 = 선택·양보 | "지급하였던 금액" (과거), "어떠한 사유이든" (선택) |

## 11. Court Document Specific Conventions (법원 제출 문서)

### 11.1 소장 작성법

#### 청구취지 작성 규칙
- 주문(판결문) 형식으로 작성하여 집행가능하게 기재
- 구체적 금액, 날짜, 행위를 명시
- "원고에게"가 아닌 "피고는 원고에게"로 시작

```
청구취지

1. 피고는 원고에게 금 100,000,000원 및 이에 대하여 2025. 6. 1.부터 이 사건
   소장 부본 송달일까지는 연 5%의, 그 다음 날부터 다 갚는 날까지는 연 12%의
   각 비율에 의한 금원을 지급하라.
2. 소송비용은 피고가 부담한다.
3. 위 제1항은 가집행할 수 있다.
라는 판결을 구합니다.
```

#### 청구원인 작성법
- 요건사실 중심으로 작성 (법률 요건에 대응하는 사실만 기재)
- 시간 순서대로 사실관계 정리
- 각 사실마다 증거를 대응시킬 수 있도록 구성
- 불필요한 감정적 표현 배제

### 11.2 준비서면 구조

```
준비서면

사건   2025가합NNNNN 손해배상(기)
원고   홍길동
피고   주식회사 OO

위 사건에 관하여 원고 소송대리인은 다음과 같이 변론을 준비합니다.

다    음

I. 쟁점의 정리
II. 원고의 주장
  1. 제1 주장
  2. 제2 주장
III. 근거
  1. 법리적 근거
  2. 사실적 근거
IV. 결론

첨부서류
1. 갑 제N호증     OOOOO     1통

2026. 3. 10.

원고 소송대리인
법무법인 OO
담당변호사 OOO
```

- 쟁점 정리 → 주장 → 근거 순서로 작성
- 상대방 주장을 인용할 때는 정확히 인용 후 반박
- 새로운 주장과 기존 주장 보충을 구분하여 기재

### 11.3 증거 표시

| 당사자 | 증거 번호 | 예시 |
|---|---|---|
| 원고 측 | 갑 제N호증 | 갑 제1호증, 갑 제2호증의 1 |
| 피고 측 | 을 제N호증 | 을 제1호증, 을 제3호증의 2 |
| 제3참가인 | 병 제N호증 | 병 제1호증 |

- 호증 번호 뒤에 세부 구분이 필요한 경우: "갑 제2호증의 1", "갑 제2호증의 2"
- 본문 인용 시: "(갑 제1호증)" 형식으로 괄호 안에 표시
- 증거설명서를 별도로 첨부하여 각 증거의 작성자, 작성일, 입증취지를 기재

## 12. Legal Term Precision (법률 용어 정확한 사용)

### 12.1 자주 혼동하는 법률 용어

| 용어 A | 용어 B | 구분 |
|---|---|---|
| 효력(效力) | 효과(效果) | 효력 = 법률행위 자체의 유효/무효 여부. 효과 = 법률행위로 인해 발생하는 결과 |
| 무효(無效) | 취소(取消) | 무효 = 처음부터 효력 없음 (당연무효). 취소 = 유효하나 취소권 행사로 소급 무효 |
| 취소(取消) | 부존재(不存在) | 취소 = 법률행위는 존재하나 하자. 부존재 = 법률행위 자체가 성립하지 않음 |
| 해제(解除) | 해지(解止) | 해제 = 소급적 효력 소멸 (매매 등). 해지 = 장래적 효력 소멸 (계속적 계약) |
| 양도(讓渡) | 이전(移轉) | 양도 = 당사자 합의에 의한 권리 이전. 이전 = 양도보다 넓은 개념 (상속 등 포함) |
| 양도(讓渡) | 승계(承繼) | 양도 = 특정 권리의 이전. 승계 = 포괄적 지위의 이전 (포괄승계/특정승계) |
| 선의(善意) | 악의(惡意) | 법률적 의미: 선의 = 어떤 사실을 모름. 악의 = 어떤 사실을 알고 있음. 도덕적 의미 아님 |
| 하자(瑕疵) | 흠결(欠缺) | 하자 = 불완전함 (하자 있는 의사표시). 흠결 = 필요한 것이 결여됨 (요건의 흠결) |
| 하자(瑕疵) | 위법(違法) | 하자 = 민사적 불완전. 위법 = 법규 위반 (행정법적 의미에서 "하자"는 위법과 유사) |
| 위반(違反) | 위배(違背) | 위반 = 규정·약속 등을 어김. 위배 = 원칙·정신 등에 어긋남 (미묘한 차이, 실무상 혼용) |
| 위반(違反) | 저촉(牴觸) | 위반 = 직접적 위반. 저촉 = 다른 규정과 충돌·모순 |
| 적용(適用) | 준용(準用) | 적용 = 해당 규정을 직접 적용. 준용 = 다른 사안에 필요한 수정을 가하여 적용 |
| 추정(推定) | 간주(看做) | 추정 = 반증으로 번복 가능. 간주 = 반증 불가, 확정적 의제 |
| 손해(損害) | 손실(損失) | 손해 = 위법행위로 인한 피해 (불법행위). 손실 = 적법행위로 인한 재산 감소 (수용 등) |
| 배상(賠償) | 보상(補償) | 배상 = 위법행위에 대한 전보 (손해배상). 보상 = 적법행위에 대한 전보 (손실보상) |
| 계약(契約) | 약정(約定) | 계약 = 채권관계 발생을 위한 합의 (법적 구속력). 약정 = 당사자 간 약속 (계약보다 넓음) |
| 계약(契約) | 합의(合意) | 계약 = 법적 효력을 가진 약속. 합의 = 의사의 합치 (법적 효력 유무와 무관할 수 있음) |
| 통지(通知) | 통보(通報) | 통지 = 상대방에게 일정한 사실을 알림 (법률적 효과 발생). 통보 = 알림 (통지보다 일상적) |
| 통지(通知) | 고지(告知) | 통지 = 일반적 알림. 고지 = 의무적·공식적 알림 (처분의 고지, 고지의무) |
| 승인(承認) | 허가(許可) | 승인 = 타인의 행위에 대한 동의. 허가 = 행정청이 금지된 행위를 해제 |
| 허가(許可) | 인가(認可) | 허가 = 자연적 자유를 회복 (영업허가). 인가 = 법률행위의 효력 보충 (합병인가) |
| 등기(登記) | 등록(登錄) | 등기 = 부동산·법인 등 (등기부). 등록 = 자동차·특허 등 (등록원부) |

### 12.2 용어 사용 시 주의사항

- 법률 용어는 일상 용어와 의미가 다를 수 있으므로 법률적 의미를 정확히 파악하여 사용
- 같은 용어라도 법 분야에 따라 의미가 달라질 수 있음 (예: "하자"는 민법과 행정법에서 뉘앙스 차이)
- 정확한 용어 사용이 법적 효력에 직접 영향을 미칠 수 있음 (예: "해제" vs "해지")

## 13. Sentence Structure Patterns (문장 구조 패턴)

### 13.1 조건문
```
~하는 경우에는 ...한다.
~한 때에는 ...한다.
~하는 경우에 한하여 ...할 수 있다.
```
- 예: "을이 대금을 기한 내에 지급하지 아니하는 경우에는, 갑은 본 계약을 해제할 수 있다."

### 13.2 의무 표현
```
~하여야 한다.
~할 의무가 있다.
~하여야 할 책임이 있다.
```
- 예: "수임인은 위임인에게 사무처리 상황을 보고하여야 한다."

### 13.3 권리 표현
```
~할 수 있다.
~할 권리가 있다.
~을(를) 청구할 수 있다.
```
- 예: "원고는 피고에게 손해배상을 청구할 수 있다."

### 13.4 예외 표현
```
다만, ~하는 경우에는 그러하지 아니하다.
다만, ~한 때에는 이 한도에서 그러하지 아니하다.
~의 경우를 제외하고는 ...한다.
```
- 예: "을은 갑의 동의 없이 본건 채권을 양도할 수 없다. 다만, 갑이 서면으로 동의한 경우에는 그러하지 아니하다."

### 13.5 정의 표현
```
~이란 ~을(를) 말한다.
~이라 함은 ~을(를) 의미한다.
~에서 "OO"이란 ~을(를) 말한다.
```
- 예: "이 약관에서 '서비스'란 회사가 제공하는 모든 온라인 서비스를 말한다."

### 13.6 준용 표현
```
~에 관하여는 ~의 규정을 준용한다.
~에 관하여 필요한 사항은 ~의 예에 따른다.
```
- 예: "감사의 선임 및 해임에 관하여는 제15조의 규정을 준용한다."

### 13.7 인용 표현
```
~에 따르면, ...
~에 의하면, ...
~에 비추어 보면, ...
~에 의하여 ...
~에 근거하여 ...
```
- 예: "대법원 판례에 의하면, 불법행위의 성립에는 고의 또는 과실이 필요하다."

### 13.8 결론 표현
```
따라서 ~한다.
이상과 같이 ~이므로, ...
이상의 이유로 ~한다.
위와 같은 이유로 ~하여야 한다.
그러므로 ~하는 것이 타당하다.
```
- 예: "이상과 같은 이유로, 피고의 손해배상 의무가 인정된다."

## 14. Paragraph Structure (단락 구성)

### 14.1 기본 원칙
- **한 단락 = 하나의 논점**: 하나의 단락에는 하나의 주제만 다룬다
- **주제문 → 뒷받침 문장 → 소결론**: 단락의 첫 문장에서 핵심 주장을 제시하고, 이를 논증한 뒤, 간결하게 정리한다
- **단락 길이**: 5~10문장이 적정. 3문장 미만이면 병합 검토, 15문장 이상이면 분리 검토

### 14.2 법률 문서의 논증 구조 (IRAC 변형)

```
법리 (Issue/Rule)  → 관련 법률 규정 및 판례 법리 제시
       ↓
사실 (Application) → 구체적 사실관계를 법리에 포섭
       ↓
포섭 (Subsumption) → 사실이 법률 요건에 해당하는지 분석
       ↓
결론 (Conclusion)  → 법적 효과 도출
```

예시:
> 「민법」 제750조에 의하면, 고의 또는 과실로 인한 위법행위로 타인에게 손해를 가한 자는 그 손해를 배상할 책임이 있다(법리). 피고는 2025. 1. 15. 차량을 운행하면서 전방주시의무를 게을리하여 원고의 차량을 추돌하였다(사실). 피고의 전방주시의무 위반은 과실에 의한 위법행위에 해당하고, 이로 인하여 원고에게 차량 수리비 상당의 손해가 발생하였다(포섭). 따라서 피고는 원고에게 위 손해를 배상할 책임이 있다(결론).

### 14.3 단락 간 전환 표현

| 관계 | 전환 표현 |
|---|---|
| 추가 | 또한, 나아가, 아울러, 더불어, 이에 더하여 |
| 대조·반박 | 그러나, 반면에, 이에 반하여, 한편, 이와 달리 |
| 인과 | 따라서, 그러므로, 이에 따라, 그 결과 |
| 예시 | 예컨대, 구체적으로, 이를테면, 가령 |
| 전환 | 한편, 다음으로, 이와 별도로, 이하에서는 |
| 요약 | 요컨대, 결국, 이상을 종합하면, 위에서 살펴본 바와 같이 |

## 15. Korean Legal Writing Quality Checklist (품질 점검 체크리스트)

### 15.1 문법 및 문장 점검
- [ ] **주어-서술어 일치**: 모든 문장에서 주어와 서술어가 정확히 호응하는가
- [ ] **문장 길이**: 3줄(약 150자) 이상인 문장은 분리를 검토하였는가
- [ ] **수동태 과다**: 불필요한 수동태가 남용되지 않았는가 (능동태 우선)
- [ ] **동일 어미 반복**: 같은 어미(~한다, ~이다 등)가 연속 3회 이상 반복되지 않는가
- [ ] **불필요한 수식어 제거**: "매우", "상당히", "다소" 등 법률 문서에 부적합한 수식어가 없는가
- [ ] **이중 피동 제거**: "되어지다", "보여지다" 등 이중 피동이 없는가
- [ ] **번역투 제거**: 영어 직역체 표현이 없는가

### 15.2 법률 용어 점검
- [ ] **용어 정확성**: 법률 용어가 정확한 의미로 사용되었는가 (해제/해지, 추정/간주 등)
- [ ] **한자 병기 일관성**: 한자 병기를 시작했으면 동일 기준으로 일관되게 적용하였는가
- [ ] **약어·정의어 일관성**: 정의한 약어를 이후 일관되게 사용하였는가

### 15.3 구조 점검
- [ ] **논리적 흐름**: 법리 → 사실 → 포섭 → 결론의 흐름이 자연스러운가
- [ ] **단락 구분**: 하나의 단락에 하나의 논점만 포함되어 있는가
- [ ] **전환 표현**: 단락 간 연결이 자연스러운가

### 15.4 형식 점검
- [ ] **번호 체계 일관성**: 조항호목 또는 I/1/가/(1) 체계가 일관되는가
- [ ] **인용 형식**: 법령, 판례 인용 형식이 정확한가
- [ ] **날짜 형식**: YYYY년 MM월 DD일 또는 YYYY. M. D. 형식이 통일되었는가
- [ ] **문체 통일**: 한 문서 내에서 ~한다체와 ~합니다체가 혼용되지 않았는가

## 16. Document-Type Specific Writing Tips (문서 유형별 작성 지침)

### 16.1 의견서 (Legal Opinion / Memorandum)

**구조**: 쟁점 → 결론 → 분석 (결론선행형)
```
I. 쟁점
   본건의 쟁점은 ~인지 여부입니다.

II. 결론
   ~한 것으로 판단됩니다.

III. 분석
   1. 관련 법령
   2. 관련 판례
   3. 본건에의 적용

IV. 유의사항 / 리스크
```

**작성 원칙**:
- 결론은 명확하고 단정적으로 기술 ("~한 것으로 판단됩니다", "~할 수 있을 것입니다")
- 불확실한 부분은 리스크 섹션에서 별도로 다룸
- 의뢰인이 이해할 수 있도록 전문 용어에 적절한 설명 추가
- 반대 논거도 함께 검토하여 균형 잡힌 분석 제공

### 16.2 소장 (Complaint)

**구조**:
```
소    장

원 고  [성명/명칭]
       [주소]
       [소송대리인 변호사 OOO]

피 고  [성명/명칭]
       [주소]

손해배상(기) 청구의 소

청 구 취 지
  [주문 형식으로 기재]

청 구 원 인
  [요건사실 중심 기재]

입 증 방 법
  [증거 목록]

첨 부 서 류
  [첨부 서류 목록]
```

**작성 원칙**:
- 청구취지는 주문(판결문) 형식으로 작성 — 그대로 주문에 옮길 수 있게
- 사실관계는 시간순으로 정리
- 법률 주장은 간결하게 (상세 법률 주장은 준비서면에서)
- 감정적 표현 배제, 객관적 사실 중심

### 16.3 결의서 (Resolution)

**구조**:
```
이 사 회 결 의 서

1. 일시: 2026년 3월 10일
2. 장소: 서울특별시 OO구 OO로 OO, 본점 회의실
3. 출석 이사: OOO, OOO, OOO (총 N인 중 N인 출석)

제1호 의안: [의안 제목]
  결의사항: [구체적 결의 내용]
  결의결과: 출석 이사 전원 찬성으로 가결

제2호 의안: [의안 제목]
  결의사항: [구체적 결의 내용]
  결의결과: 찬성 N인, 반대 N인으로 가결(부결)

위와 같이 결의하였음을 확인합니다.

2026년 3월 10일

의장  [성명]  (인)
이사  [성명]  (인)
이사  [성명]  (인)
```

**작성 원칙**:
- 의안별로 결의사항을 명확하고 구체적으로 기재
- 결의 요건 충족 여부 확인 (출석 정족수, 의결 정족수)
- 각 의안의 찬반 수를 정확히 기록
- 부대조건이 있으면 명시

### 16.4 규정류 (Internal Regulations / Bylaws)

**구조**:
```
제1장 총칙
  제1조 (목적) 이 규정은 ~을(를) 목적으로 한다.
  제2조 (정의) 이 규정에서 사용하는 용어의 정의는 다음과 같다.
    1. "OO"이란 ~을(를) 말한다.
  제3조 (적용범위) 이 규정은 ~에 적용한다.

제2장 [내용]
  제4조 ...

부칙
  제1조 (시행일) 이 규정은 YYYY년 MM월 DD일부터 시행한다.
```

**작성 원칙**:
- 조문 형식 엄격 준수 (조-항-호-목)
- 총칙(목적, 정의, 적용범위) → 본칙 → 벌칙/보칙 → 부칙 순서
- 하나의 조문에는 하나의 규범 사항만 규정
- 정의 조항에서 모든 핵심 용어를 정의
- "~한다"체 사용 (규범적 문체)
- 다른 규정과의 관계를 명확히 (우선 적용, 준용 등)
