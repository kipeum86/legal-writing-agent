# International English Legal Writing Style Guide (style-guide-en-intl)

## Scope
For documents not tied to US or UK jurisdiction: international arbitration, cross-border transactions, UN/UNCITRAL, ICC proceedings.

## Key Conventions

| Feature | Convention |
|---|---|
| Page size | A4 |
| Register | Neutral formal English; avoid jurisdiction-specific idioms |
| Numbering | Article → Section → Paragraph → (a), (b), (c) |
| Citation | Neutral format; no Bluebook or OSCOLA required |
| Date | DD Month YYYY or YYYY-MM-DD (ISO 8601) |
| Font | Times New Roman or Arial, 12pt |
| Margins | 2.5cm all sides |
| Spelling | Consistent (either US or UK, not mixed); default UK |

## Register Rules
- Avoid US-specific terms: "attorney" → "counsel"; "brief" → "submission"
- Avoid UK-specific terms: "barrister", "solicitor" → "counsel", "legal representative"
- Use UNCITRAL/ICC terminology where applicable
- Plain, clear language preferred

## Citation
- Treaties: Full title, date, reference number
- Arbitral awards: [Institution] Case No. [Number], Award of [Date]
- No mandatory citation manual; consistency is key
