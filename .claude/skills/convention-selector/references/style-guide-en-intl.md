# International English Legal Writing Style Guide (style-guide-en-intl)

## Scope
Use this guide for documents not anchored to a single domestic common-law convention, including cross-border advice, international arbitration, treaty-based analysis, multinational compliance work, UN / UNCITRAL / ICC style materials, and neutral legal memoranda intended for mixed audiences.

## Core Positioning
- Aim for neutral, institution-facing legal English
- Avoid unnecessary US-only or UK-only drafting habits unless the user or forum specifically requests them
- Prefer clarity, portability, and cross-border intelligibility over local courtroom idiom
- Keep spelling internally consistent; default to UK spelling unless the matter context clearly points elsewhere

## Key Conventions

| Feature | Convention |
|---|---|
| Page size | A4 |
| Register | Neutral formal English; avoid jurisdiction-specific idioms |
| Numbering | Article → Section → Paragraph → (a), (b), (c) |
| Citation | Neutral format; no mandatory Bluebook or OSCOLA overlay unless requested |
| Date | DD Month YYYY or YYYY-MM-DD (ISO 8601) |
| Font | Times New Roman or Arial, 12pt |
| Margins | 2.5cm all sides |
| Spelling | Consistent throughout; default UK |

## Register Rules
- Avoid US-specific role labels where a neutral term exists: `attorney` → `counsel`
- Avoid UK-specific profession labels where a neutral term exists: `barrister`, `solicitor` → `counsel` or `legal representative`
- Prefer neutral verbs such as `provides`, `requires`, `permits`, `prohibits`, `states`
- Avoid domestic procedural shorthand unless the audience is clearly domestic
- Plain, clear language is preferred over traditional legal ornament

## Terminology Choices
| Domestic Term | Neutral International Preference |
|---|---|
| attorney | counsel |
| brief | submission or memorandum |
| claimant / plaintiff | claimant, applicant, or party-specific label based on forum |
| defendant / respondent | respondent or party-specific label based on forum |
| exhibit | annex, appendix, or schedule |

## Numbering and Structure

### Preferred hierarchy
```
Article 1
Section 1.1
Paragraph 1.1.1
(a)
(i)
```

### Rules
- Keep numbering decimal and transparent
- Do not mix domestic labels (`Clause`, `Section`, `Article`) without a reason tied to the forum or precedent
- Use `Article` for top-level normative or institutional documents
- Use `Section` for memorandum-style analytical writing if needed
- Use `(a), (b), (c)` and `(i), (ii), (iii)` for nested analytical points

## Document Structure Guidance

### Cross-border memorandum
1. Instructions or question presented
2. Executive summary
3. Facts / assumptions
4. Applicable framework
5. Analysis
6. Conclusion
7. Limitations / reservations

### International submission or comment letter
1. Introduction and interest of the submitting party
2. Summary position
3. Legal and factual framework
4. Detailed submissions
5. Requested outcome

### Arbitration-focused analysis
1. Background
2. Issues
3. Applicable instruments / rules
4. Jurisdiction / admissibility if relevant
5. Merits analysis
6. Relief or recommendation requested by the user

## Citation

### General rule
- Use a stable, descriptive format that identifies the authority clearly without importing a domestic citation system unless the user requests one

### Treaties and conventions
Format:
```text
Full Instrument Title, opened for signature Date, Reference
```

### Arbitral awards
Format:
```text
[Institution] Case No [Number], Award of [Date]
```

### National laws in international documents
- Name the jurisdiction explicitly on first reference
- Use the jurisdiction's official statute name where available
- Avoid Bluebook-only or OSCOLA-only abbreviations unless the entire document follows that manual

### Institutional materials
- Use the issuing institution, instrument title, date, and identifying number if available
- For UNCITRAL, ICC, OECD, EU, UN, and agency documents, preserve official titles

## Defined Terms and Multilingual Handling
- Define cross-border party labels clearly on first use
- Use translated terms together with the original-language term where needed for legal precision
- If a domestic concept has no neat equivalent, use a neutral English label and preserve the original in parentheses on first reference
- Do not alternate between domestic labels once a neutral label has been chosen

## Dates, Currency, and Formatting Choices
- Use `DD Month YYYY` unless ISO dates are operationally better for tables and timelines
- Identify currency with code or full unit where ambiguity exists
- Use A4 page setup and neutral headings
- If the document mixes legal systems, signpost the governing framework explicitly instead of assuming a domestic reader

## Page Setup Summary
- Use A4 paper
- Use the parameters in `../../output-formatter/references/page-setup-en-uk.md` unless the user supplies a different international house style
- Default body font is Times New Roman 12pt
- Use 1.5 line spacing for prose unless the forum requires something else

## Common Drafting Corrections
| Avoid | Prefer |
|---|---|
| heavy Bluebook strings in a neutral memo | descriptive neutral citations |
| UK-only role labels in mixed-audience text | neutral role labels |
| US-only procedural shorthand | descriptive forum-neutral wording |
| mixed spelling systems | one consistent spelling system |

## Quality Checklist
- The prose is intelligible to readers from more than one legal system
- Numbering is consistent and not tied unnecessarily to one domestic habit
- Citation style is internally consistent
- Domestic jargon has been replaced with neutral equivalents where possible
- Dates, currency, and institutional names are explicit
- Spelling is consistent throughout
