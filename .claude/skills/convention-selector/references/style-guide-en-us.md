# US English Legal Writing Style Guide (style-guide-en-us)

## 1. Register

### Formal Legal Register
- "Must" for obligations (preferred per Bryan Garner's modern style)
- "Shall" for obligations only when matching existing precedent, house style, or jurisdiction convention
- "May" for permissions
- "Will" for future actions or declarations of intent
- "Is entitled to" for rights
- No contractions
- No colloquialisms
- Active voice preferred; passive acceptable only for emphasis on the object or when the actor is unknown
- First person prohibited in formal documents (acceptable in memos and client letters)

### Prohibited
- Legalese where plain English suffices (but maintain precision for terms of art)
- Informal language, slang, or rhetorical questions
- Hedging language ("it would seem," "arguably") in operative provisions

## 2. Bryan Garner's Core Principles

### 2.1 "Must" over "Shall"
"Shall" is the most litigated word in the English language. It has been interpreted as mandatory, directory, and even permissive. Modern drafting replaces it:

| Traditional | Modern Equivalent |
|---|---|
| shall | must (obligation on subject) |
| shall not | must not / may not (prohibition) |
| shall be entitled | is entitled / may (right or permission) |
| shall mean | means (definition) |
| no person shall | a person must not / a person may not |

Use "shall" only when (a) the jurisdiction mandates it, (b) house style requires it, or (c) you are amending a document that already uses "shall" consistently.

### 2.2 Active Voice
Put the actor before the action. Active voice identifies who does what, reducing ambiguity.

- Weak: "Written notice must be given by the Landlord."
- Strong: "The Landlord must give written notice."

### 2.3 Front-Load the Key Point
The most important information belongs in the first sentence of a paragraph (and the first paragraph of a section). Readers scan; do not bury the operative rule in subordinate clauses.

### 2.4 Sentence Length
Aim for an average of fewer than 25 words per sentence. No single sentence should exceed 45 words. If a sentence has more than one condition and one consequence, break it up.

### 2.5 Avoid Nominalizations
A nominalization converts a verb or adjective into a noun, weakening the sentence.

| Nominalization | Plain English |
|---|---|
| failure to comply | failing to comply / if X does not comply |
| the making of a decision | deciding |
| gave consideration to | considered |
| reached an agreement | agreed |
| is in violation of | violates |
| provide a description of | describe |
| effect a termination | terminate |
| make payment of | pay |

### 2.6 Avoid Archaic Legalese
The following words add no precision and should be eliminated in modern drafting:

| Archaic | Modern |
|---|---|
| hereby | (delete — the document itself performs the act) |
| herein | in this agreement / in this section |
| hereinafter | (use a defined term instead) |
| whereas | (delete from recitals; use "Background" heading) |
| witnesseth | (delete entirely) |
| aforementioned | (use the defined term or "the") |
| said (as adjective) | the / this / that |
| same (as pronoun) | it / the [noun] |
| hereinabove / hereinbelow | above / below |
| to wit | namely / specifically |

### 2.7 Parallel Structure
Items in a list or series must share the same grammatical form.

- Wrong: "The tenant must (a) pay rent, (b) maintenance of the premises, and (c) comply with rules."
- Right: "The tenant must (a) pay rent, (b) maintain the premises, and (c) comply with the rules."

### 2.8 Avoid Elegant Variation
Use the same word for the same concept throughout a document. Do not alternate between "employee," "worker," "staff member," and "personnel" for the same defined concept. Variation creates ambiguity about whether different words signal different meanings.

## 3. Common Legal Writing Errors

| Error | Example | Correction |
|---|---|---|
| "Shall" ambiguity | "The report shall be filed." | "The applicant must file the report." |
| "And/or" | "penalties and/or sanctions" | "penalties, sanctions, or both" |
| "Prior to" | "prior to the hearing" | "before the hearing" |
| "Subsequent to" | "subsequent to execution" | "after execution" |
| "In the event that" | "in the event that default occurs" | "if a default occurs" |
| "Such" as demonstrative | "such agreement" | "the agreement" / "this agreement" |
| "Provided that" | "provided that notice is given" | "but only if notice is given" or "except that" |
| Double negative | "not unlikely" | "likely" |
| Triple negative | "shall not be invalid unless not" | rewrite entirely |
| Dangling modifier | "Having reviewed the record, the motion is granted." | "Having reviewed the record, the Court grants the motion." |
| Passive voice overuse | "It was decided that the contract was breached." | "The court decided that X breached the contract." |
| "Which" vs. "that" | "The clause which limits liability..." (if restrictive) | "The clause that limits liability..." |
| Run-on sentence | Two independent clauses joined by comma only | Use a period, semicolon, or coordinating conjunction |
| "It is important to note that" | "It is important to note that the statute requires..." | "The statute requires..." (delete the throat-clearing) |
| "Clearly" / "Obviously" | "Clearly, the defendant is liable." | "The defendant is liable because..." (state the reason) |
| "The fact that" | "Despite the fact that" | "Although" / "Even though" |
| Misplaced "only" | "The court only considered the first argument." | "The court considered only the first argument." |
| "As to" | "As to the defendant's motion..." | "Regarding the defendant's motion..." or restructure |
| "Impact" as verb | "This impacts the analysis." | "This affects the analysis." |
| "Utilize" | "utilize resources" | "use resources" |
| Buried verb | "make a determination" | "determine" |
| Surplus words | "null and void and of no further force or effect" | "void" |
| Doublets/triplets | "give, devise, and bequeath" | choose the precise single term |

## 4. Bluebook Citation — Detailed Rules

### 4.1 Full Case Citations
```
[Party 1] v. [Party 2], [Volume] [Reporter] [First Page], [Pincite] ([Court] [Year]).
```
Examples:
- Brown v. Board of Education, 347 U.S. 483, 495 (1954).
- Ashcroft v. Iqbal, 556 U.S. 662, 678 (2009).
- In re Application of XYZ Corp., 123 F.3d 456, 460 (2d Cir. 2023).
- Smith v. Jones, 45 Cal. App. 4th 123, 130 (2024).

**Rules**:
- Underline or italicize the case name (check local rules).
- Include the court in parentheses unless the reporter makes the court obvious (e.g., U.S. Reports = Supreme Court).
- Use official reporter if available; parallel citations where local rules require.

### 4.2 Short Form Citations

| Form | When to Use | Example |
|---|---|---|
| Id. | Immediately following citation to the same source | Id. at 485. |
| Id. (no pincite) | Same source, same page | Id. |
| Short case name | Same source cited previously, but not immediately preceding | Brown, 347 U.S. at 495. |
| Supra | Non-case sources cited previously in full | Garner, supra note 12, at 45. |
| Infra | Forward reference | See infra Part III.B. |

**Rules**:
- "Id." may not be used if more than one source is cited in the preceding footnote.
- Always italicize "Id." and all other signals.

### 4.3 Statutory Citations

**Federal**:
```
[Title] U.S.C. § [Section] ([Year]).
```
- 42 U.S.C. § 1983 (2024).
- 15 U.S.C. §§ 78a-78qq (2024). (use §§ for multiple sections)

**State**:
```
[State Code Abbreviation] § [Section] ([Publisher] [Year]).
```
- Cal. Civ. Code § 1708 (West 2024).
- N.Y. Gen. Bus. Law § 349 (McKinney 2024).
- Tex. Bus. & Com. Code Ann. § 17.46 (West 2024).

### 4.4 Regulatory Citations (C.F.R.)
```
[Title] C.F.R. § [Section] ([Year]).
```
- 17 C.F.R. § 240.10b-5 (2025).
- 40 C.F.R. § 122.1 (2025).

**Federal Register**:
```
[Volume] Fed. Reg. [Page] ([Date]).
```
- 89 Fed. Reg. 12345 (Feb. 15, 2024).

### 4.5 Constitutional Provisions
```
U.S. Const. [article/amendment] [section/clause].
```
- U.S. Const. art. I, § 8, cl. 3.
- U.S. Const. amend. XIV, § 1.

State:
- Cal. Const. art. I, § 7.

### 4.6 Secondary Sources

**Treatises**:
```
[Volume] [Author], [Title] § [Section], at [Page] ([Edition] [Year]).
```
- 3 William Meade Fletcher, Fletcher Cyclopedia of the Law of Corporations § 990, at 45 (perm. ed., rev. vol. 2020).

**Law Review Articles**:
```
[Author], [Title], [Volume] [Journal] [First Page], [Pincite] ([Year]).
```
- Cass R. Sunstein, On the Expressive Function of Law, 144 U. Pa. L. Rev. 2021, 2030 (1996).

**Restatements**:
```
Restatement ([Number]) of [Subject] § [Section] ([Year]).
```
- Restatement (Second) of Contracts § 90 (Am. L. Inst. 1981).
- Restatement (Third) of Torts: Liab. for Physical & Emotional Harm § 7 (Am. L. Inst. 2010).

### 4.7 Internet and Electronic Sources
```
[Author], [Title], [Source] ([Date]), [URL].
```
- SEC, Staff Accounting Bulletin No. 121 (Mar. 31, 2022), https://www.sec.gov/....

For cases retrieved from electronic databases when no official reporter page is available:
```
[Case Name], No. [Docket Number], [Database Identifier], at *[Star Page] ([Court] [Date]).
```
- Smith v. Jones, No. 22-cv-1234, 2023 WL 123456, at *3 (S.D.N.Y. Jan. 15, 2023).

### 4.8 Signals

| Signal | Meaning |
|---|---|
| [no signal] | Source directly states the proposition |
| See | Source clearly supports the proposition but does not directly state it |
| See also | Additional source support; follows a source with no signal or "see" |
| Cf. | Source supports by analogy |
| Compare ... with ... | Comparison of sources illuminates the point |
| But see | Source clearly supports the contrary proposition |
| But cf. | Source supports the contrary by analogy |
| Contra | Source directly states the contrary |
| See generally | Background or general information |

**Rules**: Italicize all signals. Capitalize only when beginning a new citation sentence. Separate signals of different types with a semicolon.

### 4.9 Explanatory Parentheticals
Place parenthetical explanations after the citation:
- Brown v. Board of Education, 347 U.S. 483, 495 (1954) (holding that racial segregation in public schools violates the Equal Protection Clause).
- Use a present participle to begin: (holding...), (finding...), (noting...), (explaining...).
- Quoting parenthetical: (quoting Smith v. Jones, 100 U.S. 200, 205 (1990)).

### 4.10 String Citations
- Order authorities within each signal by jurisdiction and court hierarchy.
- Federal: Supreme Court, then circuit courts (by circuit number), then district courts.
- State: highest court first; alphabetical by state when citing multiple states.
- Separate citations within a string with semicolons.
- Use string citations sparingly; prefer the most authoritative source with explanation.

### 4.11 Pinpoint Citations
- Pages: Id. at 485.
- Paragraphs: Id. ¶ 12.
- Sections: Id. § 3.
- Multiple pages: 347 U.S. at 483-85 (drop repetitive digits per Bluebook rule 3.2(a)).
- Multiple paragraphs: Id. ¶¶ 12-15.

## 5. Memo/Brief Writing Structure

### 5.1 IRAC Method (Deep Dive)

**Issue**: Frame the legal question precisely. A good issue statement names the legal standard and the key facts.
- Weak: "Is the defendant liable?"
- Strong: "Whether the landlord breached the implied warranty of habitability by failing to repair a persistent roof leak for six months after receiving written notice."
- Begin with "Whether" (memo) or a persuasive framing (brief).

**Rule**: Synthesize the governing rule from multiple authorities. Start with the general rule, then refine with specific elements or factors.
- General rule → elements/factors → exceptions → burden of proof.
- Cite the foundational authority for each proposition.
- Show how courts have interpreted ambiguous elements.

**Application**: Apply each element of the rule to the specific facts of your case.
- Dedicate at least one paragraph to each disputed element.
- Use analogical reasoning: compare your facts to the facts of precedent cases.
- Address counterarguments within the application section.
- Use "here" to pivot from the rule to the facts: "Here, the landlord received written notice on January 5..."

**Conclusion**: State the likely outcome clearly. In a memo, be balanced. In a brief, be definitive.

### 5.2 CRAC Variant (Conclusion-First)
For busy readers (partners, judges), lead with the conclusion:
1. **Conclusion**: State the answer up front.
2. **Rule**: Present the legal framework.
3. **Application**: Apply facts to law.
4. **Conclusion**: Restate with nuance.

Use CRAC for office memos, executive summaries, and motion briefs. Use IRAC for exam-style analysis and detailed research memos.

### 5.3 Point Headings
Point headings in briefs must be:
- Full, persuasive sentences (not labels or phrases).
- Legally significant — they should state the legal conclusion you want the court to reach.
- Fact-specific — they should reference the actual facts of the case.

| Weak | Strong |
|---|---|
| "Standard of Review" | "This Court Reviews the Grant of Summary Judgment De Novo." |
| "Breach of Contract" | "Defendant Breached the Supply Agreement by Failing to Deliver 10,000 Units by the March 1 Deadline." |
| "Standing" | "Plaintiff Has Standing Because It Suffered a Concrete Financial Injury Traceable to Defendant's Conduct." |

### 5.4 Statement of Facts
**Memorandum** (neutral/objective):
- Chronological organization.
- Include all material facts, favorable and unfavorable.
- Do not argue or characterize.
- Cite to the record for each factual assertion.

**Brief** (persuasive):
- Organize to emphasize favorable facts (chronological is still common, but thematic is acceptable).
- Use detail and specificity for favorable facts; state unfavorable facts accurately but without emphasis.
- Never misstate or omit a material fact.
- Use active voice for your client's actions; passive voice (sparingly) to de-emphasize the opposing party's favorable facts.

### 5.5 Roadmap Paragraphs
The first paragraph of a Discussion or Argument section should:
1. State the overall conclusion.
2. Identify the governing legal standard.
3. Preview the sub-arguments in the order they will appear.

Example: "The Court should grant summary judgment because Plaintiff cannot establish the three elements of a prima facie case under the Jones Act: (1) duty, (2) breach, and (3) causation. Each element fails for an independent reason."

## 6. Sentence-Level Craft

### 6.1 Topic Sentences That Do Legal Work
Every paragraph should open with a sentence that advances the analysis, not merely introduces a topic.

- Weak: "The next issue is causation."
- Strong: "The plaintiff cannot establish causation because no evidence links the spill to the contamination found three miles downstream."

### 6.2 Transition Words and Phrases

| Function | Transitions |
|---|---|
| Addition | Moreover, Furthermore, In addition, Similarly, Likewise |
| Contrast | However, Nevertheless, By contrast, Conversely, On the other hand, Yet |
| Cause/Effect | Therefore, Accordingly, As a result, Consequently, Thus, Hence |
| Example | For example, For instance, Specifically, In particular, Notably |
| Concession | Although, Even though, Notwithstanding, Granted, Admittedly |
| Sequence | First...Second...Third, Initially, Subsequently, Finally |
| Emphasis | Indeed, In fact, Significantly, Importantly |
| Summary | In sum, To summarize, In short, Ultimately |

### 6.3 Emphasis Techniques
- **Short sentence after long**: A long sentence explaining a complex rule, followed by a short declarative sentence, creates emphasis. "That standard is not met here."
- **Position**: The most emphatic positions in a sentence are the beginning and the end. Place the most important information there.
- **Periodic sentence**: Build suspense by placing the main clause at the end: "Despite three written warnings, two formal complaints, and a final demand letter, the landlord took no action."

### 6.4 Paragraph Length
- Aim for 3-6 sentences per paragraph.
- Single-sentence paragraphs are acceptable for emphasis but should be rare.
- If a paragraph exceeds 8 sentences, consider splitting it.

### 6.5 Dovetailing
End one paragraph with a concept and begin the next paragraph by picking up that concept. This creates cohesion.

- End of paragraph: "...the court emphasized the importance of the employer's knowledge."
- Start of next: "That knowledge requirement is satisfied here because..."

## 10. Defined Terms
- First use: full name with parenthetical abbreviation.
- Pattern: "XYZ Corporation (the 'Company')" or "XYZ Corporation ('XYZ')."
- Subsequent: defined term only.
- Capitalize defined terms consistently throughout.
- Never vary the form of a defined term (do not alternate between "Company" and "Corporation" if "Company" is the defined term).
- If a defined term is used fewer than three times, consider whether a definition is necessary.

## 12. Legal Writing Quality Checklist

Use this checklist to review any legal document before finalizing:

### Voice and Style
- [ ] Active voice used at least 70% of the time
- [ ] Average sentence length is under 25 words
- [ ] No sentence exceeds 45 words
- [ ] No nominalizations where a verb would work

### Word Choice
- [ ] "Must" used for obligations (or "shall" used consistently if traditional style)
- [ ] No "and/or" — rephrased with "or both"
- [ ] No archaic legalese (hereby, herein, hereinafter, whereas, witnesseth)
- [ ] No throat-clearing phrases ("It is important to note that," "It should be noted")
- [ ] No elegant variation — same concept uses same word throughout
- [ ] "Which" / "that" used correctly (restrictive = "that"; nonrestrictive = "which" + comma)

### Structure
- [ ] Each paragraph opens with a topic sentence that advances the analysis
- [ ] Paragraphs are 3-6 sentences long
- [ ] Point headings (in briefs) are full, persuasive, fact-specific sentences
- [ ] Roadmap paragraph appears at the start of Discussion/Argument sections

### Citations
- [ ] All legal propositions are supported with citations
- [ ] No string citations without a clear purpose
- [ ] Pinpoint citations used for all specific propositions
- [ ] Short forms used correctly (Id. only when immediately following the same source)
- [ ] Signals (see, see also, cf.) used correctly and italicized

### Defined Terms
- [ ] Every defined term is introduced on first use with a parenthetical
- [ ] Defined terms are capitalized consistently
- [ ] No defined term appears in the document without being defined
- [ ] Defined terms are not varied (no elegant variation)

### Formatting
- [ ] Margins, font, and spacing conform to applicable rules
- [ ] Page numbers present
- [ ] Table of Contents and Table of Authorities included (if required)
- [ ] No orphan footnotes or widowed headings
