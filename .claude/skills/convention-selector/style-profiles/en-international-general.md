# Style Profile: en-international-general

Use this compact profile for international English general legal documents where no jurisdiction-specific citation system controls.

## Load Scope

- Load this file for English international general documents.
- Do not load US, UK, Korean, or Mode A-D references unless requested.

## Core Conventions

- Register: neutral formal English.
- Citations: preserve user-provided source labels verbatim; use neutral pinpoint references when supplied.
- Page setup: A4, 2.54 cm margins, Times New Roman 12 pt unless house style overrides.
- Numbering: article/section/paragraph hierarchy selected by template.
