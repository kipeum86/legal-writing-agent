# Style Profile: ko-korea-advisory

Use this compact profile for Korean legal opinions, legal review opinions, and client memoranda.

## Load Scope

- Load this file at D2 for Korean advisory documents.
- Load `docs/_private/ko-legal-opinion-style-guide.md` as an optional mandatory supplement when it exists locally.
- Do not load English/US, English/UK, international, or Mode A-D references unless the user explicitly requests them.

## Core Conventions

- Structure: matter heading, scope/facts, issues, conclusion placeholders or user-supplied conclusions, analysis, closing limitation.
- Register: formal Korean advisory style. `합니다` style is permitted for client-facing opinions when consistent across the document.
- Citations: Korean statutes use `「법률명」 제N조 제N항`; Korean cases use `대법원 YYYY. MM. DD. 선고 YYYY다NNNNN 판결`.
- Numbering: opinion sections generally use `I.`, `1.`, `가.`, `(1)`, `(가)` unless the private style guide overrides.
- Page setup: A4; body 12 pt; CJK font settings follow selected house/private style.

## Drafting Boundary

Advisory documents are Conditional. Without an authority packet, produce a skeleton draft with `[Authority needed: ...]`, `[Counsel conclusion needed: ...]`, `[Counsel certainty needed: ...]`, and `[Counsel risk assessment needed: ...]`.
