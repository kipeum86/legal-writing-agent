# Template: Litigation (Litigation Documents) — English

## Use This Template For
- Complaints and petitions
- Answers and responsive pleadings
- Motions and supporting briefs
- Mediation statements
- Arbitration requests or statements of claim

## Support Profile
- Support level: `Conditional`
- A full filing requires claims or defenses, procedural posture, governing authorities, and a factual record
- If those inputs are incomplete, keep the pleading structure and use placeholders rather than inventing allegations, counts, or authorities

## Section Tags
- `[B]` = Required procedural or filing framework
- `[S]` = Matter-specific allegations, defenses, or relief
- `[AP]` = Authority packet required for a supportable section

## Variant A — Complaint / Petition

1. `Caption` `[B/S]`
   - Court or tribunal name
   - Party names
   - Case number if known
   - Title of pleading

2. `Parties` `[S]`
   - Identify each plaintiff and defendant or petitioner and respondent
   - Include residence, organization, or status details only if relevant to jurisdiction or the claim

3. `Jurisdiction and Venue` `[S/AP]`
   - State the basis for subject-matter jurisdiction
   - State the basis for personal jurisdiction and venue where required

4. `Factual Allegations` `[S]`
   - Present facts chronologically or by claim theme
   - Use numbered paragraphs if the forum expects pleading-style format

5. `Counts / Causes of Action` `[S/AP]`
   - `Count I — [Claim Name]`
   - `A. Elements or governing rule`
   - `B. Application to pleaded facts`
   - `C. Harm or relief basis`
   - Repeat for each claim

6. `Prayer for Relief` `[S/AP]`
   - Damages
   - Declaratory or injunctive relief
   - Costs, interest, attorneys' fees if supported

7. `Jury Demand` `[B/S]` if applicable
8. `Signature Block` `[B/S]`
9. `Verification / Certificate / Exhibits` `[B/S]` if required by the forum

## Variant B — Answer / Responsive Pleading

1. `Caption` `[B/S]`
2. `Preliminary Statement` `[S]` if strategically useful
3. `Responses to Allegations` `[S]`
   - Admit, deny, or state lack of knowledge paragraph by paragraph
4. `Affirmative Defenses` `[S/AP]`
   - State each defense separately
   - Include supporting legal basis if the forum or strategy requires it
5. `Counterclaims / Crossclaims` `[S/AP]` if applicable
6. `Prayer for Relief` `[S]`
7. `Signature / Certificate of Service` `[B/S]`

## Variant C — Motion / Brief / Memorandum of Law

1. `Caption and Motion Title` `[B/S]`
2. `Relief Requested` `[S]`
3. `Introduction` `[S]`
4. `Statement of Relevant Facts` `[S]`
5. `Standard of Review / Governing Law` `[AP]`
6. `Argument` `[S/AP]`
   - `A. Point Heading`
   - `1. Governing Rule`
   - `2. Application`
   - `3. Response to opposing position`
7. `Conclusion` `[S]`
8. `Signature / Filing Certifications` `[B/S]`

## Variant D — Mediation Statement / Arbitration Request

1. `Caption or Submission Heading` `[B/S]`
2. `Parties and Background` `[S]`
3. `Issues in Dispute` `[S]`
4. `Contractual / Procedural Basis for the Proceeding` `[AP]`
5. `Claimant Position` `[S/AP]`
6. `Respondent Position` `[S/AP]` if the document is bilateral or summary in form
7. `Relief Sought` `[S]`
8. `Supporting Materials` `[S]`
9. `Signature Block` `[B/S]`

## Boilerplate vs Substantive Guidance
- Boilerplate: captions, signature blocks, certificates, exhibit lists, service blocks
- Substantive: jurisdiction, factual allegations, counts, defenses, argument headings, relief sought
- Conditional support warning: never fabricate counts, defenses, standards of review, or requested relief

## Drafting Reminders
- Use local court, arbitral, or tribunal rules over this template whenever a forum-specific format is provided
- Maintain tight alignment between factual allegations and each claim or defense
- For motions and briefs, the argument headings should convey the conclusion, not merely label the topic
