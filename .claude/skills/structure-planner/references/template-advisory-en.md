# Template: Advisory (Memorandum) — English

## Standard Structure (IRAC/CRAC)

```
1. Header
   - TO: [Recipient]
   - FROM: [Author]
   - DATE: [Date]
   - RE: [Subject]

2. Executive Summary / Brief Answer
   - Concise answer to each issue

3. Statement of Facts
   - Relevant factual background
   - Key dates and events

4. Issues Presented
   - Issue 1: [Statement]
   - Issue 2: [Statement]

5. Discussion / Analysis
   5.1 Issue 1
     5.1.1 Applicable Rule
     5.1.2 Analysis / Application
     5.1.3 Counter-arguments (if applicable)
     5.1.4 Conclusion
   5.2 Issue 2
     (same structure)

6. Conclusion / Recommendations

7. Assumptions and Limitations

8. Appendices (if applicable)
```

## Notes
- English advisory follows IRAC: Issue → Rule → Application → Conclusion
- Or CRAC: Conclusion → Rule → Application → Conclusion
- Support level: Conditional — requires authority packet for sections 5+
