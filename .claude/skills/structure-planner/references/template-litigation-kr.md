# Template: Litigation (소송문서) — Korean

## Complaint (소장)
```
소        장

원  고  [원고 성명/상호]
        [주소]
        [대리인: 법무법인 OO, 담당변호사 OOO]

피  고  [피고 성명/상호]
        [주소]

[사건명] 청구의 소

청 구 취 지
1. 피고는 원고에게 금 OOO원 및 이에 대하여...을 지급하라.
2. 소송비용은 피고의 부담으로 한다.
3. 제1항은 가집행할 수 있다.
라는 판결을 구합니다.

청 구 원 인
1. 당사자의 지위
2. 사실관계
3. 관련 법리
4. 결론

입 증 방 법
1. 갑 제1호증: [증거 설명]
2. 갑 제2호증: [증거 설명]

첨 부 서 류
1. 위 입증방법  각 1통
2. 소장부본     1통
3. 송달료납부서 1통

YYYY년 MM월 DD일

원고 [성명/상호]
(대리인 변호사 OOO)

OO지방법원 귀중
```

## Answer (답변서) / Preparatory Brief (준비서면)
```
답변서 (또는 준비서면)

사건번호: YYYY가합NNNNN [사건명]
원  고: [성명]
피  고: [성명]

1. 청구취지에 대한 답변 (답변서의 경우)
2. 청구원인에 대한 인부
3. 피고의 주장
   가. 항변 1
   나. 항변 2
4. 관련 법리 및 판례
5. 결론

YYYY년 MM월 DD일
피고 대리인 변호사 OOO
OO지방법원 제O민사부 귀중
```

## Notes
- Support level: Conditional — requires claims/defenses + applicable rules + authorities
- Strict formatting per court filing requirements
- 청구취지 must be specific and actionable
