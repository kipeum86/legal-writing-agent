# Template: Advisory (의견서) — Korean

## Standard Structure

```
1. 표지
   - 문서 제목 (법률의견서 / 법률검토의견)
   - 수신인
   - 발신인
   - 날짜
   - 참조번호 (있는 경우)

2. 요약 (Executive Summary)
   - 검토 배경
   - 핵심 결론 요약

3. 쟁점 정리
   - 쟁점 1: [제목]
   - 쟁점 2: [제목]
   - ...

4. 결론 (쟁점별)
   - 쟁점 1에 대한 결론
   - 쟁점 2에 대한 결론
   - ...

5. 검토 / 분석
   5.1 쟁점 1
     5.1.1 관련 법령
     5.1.2 판례 및 학설
     5.1.3 사안에의 적용
     5.1.4 소결론
   5.2 쟁점 2
     (동일 구조)

6. 전제사항 및 한계
   - 검토의 전제
   - 사실관계 확인 범위
   - 의견의 한계

7. 첨부
   - 관련 법령 발췌
   - 참고자료 목록
```

## Notes
- Korean advisory follows 쟁점→결론→분석 order (conclusion before analysis)
- This differs from English IRAC/CRAC structure
- Each issue (쟁점) should have: applicable law → precedent → application → sub-conclusion
- Support level: Conditional — requires authority packet for sections 5+
