# Template: Corporate (기업문서) — Korean

## Board Resolution (이사회 결의서)
```
1. 제목: 이사회 결의서
2. 회사명 / 기본사항
3. 일시 및 장소
4. 출석 이사 / 감사
5. 의안
   제1호 의안: [제목]
     - 내용
     - 결의사항
   제2호 의안: [제목]
     - 내용
     - 결의사항
6. 기타 사항
7. 폐회
8. 날짜
9. 서명
   의장: [성명] (인)
   이사: [성명] (인)
   감사: [성명] (인)
```
